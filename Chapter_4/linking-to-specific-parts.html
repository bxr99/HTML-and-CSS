Computer Science Terms

What is an algorithm?
What is meant by algorithmic complexity?
What are data structures?

Algorithms

Simply put, an algorithm is a method or process followed to solve a problem.

Algorithmic Complexity

Algorithmic complexity is concerned about how fast or slow particular algorithms performs.

Data-Structures

Data structures are ways to organize data (information).
Some examples of data structures are:

  • Arrays: a collection of data items of the same type stored contiguously
  • Linked Lists: a sequence of data items, where each one points to the next one
