The strong Tag

The use of the strong tag indicates that its content is important. By default browsers will show the contents between a strong element as bold.

The em Tag

The use of the em tag indicates emphasis (obviously..) of certain text. By default browsers will show the contents between a em element in italics.

The blockquote and q Tag

blockquote and q tags are the two commonly used elements for marking up quotations. The blockquote is used for longer quotes that take up an entire paragraph. The q tag is used for shorterr quotes that sit within a paragraph. Like the founding fathers once said, Ball is life.

Here I am using blockquote tags, note that when using this element that browsers tend to indent the contents automatically. This tag should not be used to indent, CSS is to be the preferred choice
